2. How to download to computer
    a. run the command in (cmd) to install esptool
        pip install esptool
    b. install mpremote. run in (cmd) the command
        pip install mpremote
    c. add a mpremote path to the computer:
        win -> environment variables -> (double press) path -> new [paste there the text in the following line]
        change(username, and disk if needed (C:)) 
            C:\Users\[username]\AppData\Local\Packages\PythonSoftwareFoundation.Python.3.11_qbz5n2kfra8p0\LocalCache\local-packages\Python311\Scripts
        save. press-
            OK -> OK -> OK
        check if it is working(paste in CMD)
            mpremote --help

3. How to burn the esp32D code into him
    a. delete the esp32d memory [change the port into the right one]
        a. press (win x) and  go to device manager. 
            go to ports and check in which port the esp is connected.(for example: "COM6")
        b. in the following code, change the (port) into the right one
            python3 -m esptool --chip esp32 --port (port) erase_flash
    b. downloadthe micropython code
        go to the website:
            https://micropython.org/download/ESP32_GENERIC/
                go to download
                    go to Firmware
                        go to Releases
                            download the link:
                                v1.26.1 (2025-09-11) .bin
    c. upload the micropython to the esp32D 
        a. change the (port) and the (user), then run in (cmd)
            python3 -m esptool --chip esp32 --port (PORT) --baud 460800 write-flash -z 0x1000 "C:\Users\(user)\Downloads\ESP32_GENERIC-20250911-v1.26.1.bin"
        b. check if it is working [change (port), then run in (cmd)]
            mpremote connect (port) repl
                print("Hello from ESP32!")
